# pretoughbreak32playerscoreboard

32 player scoreboard modification for Team Fortress 2, but it looks like the scoreboard from before the Tough Break update back on 12/17/2015.

Files go in: steam -> steamapps -> common -> Team Fortress 2 -> tf -> custom -> my_custom_stuff

I think the "my_custom_stuff" folder can be renamed to whatever you want, and it should still work.

This scoreboard mod also fixes numerical ping values getting cut off when you have over 99 ping.

I do not remember where I originally got the file, but I'm glad I found it regardless, as I enjoyed the way the scoreboard used to look before the Tough Break update.
